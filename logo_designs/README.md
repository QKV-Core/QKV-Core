# QKV Core Logo Files

## Logo Dosyası Konumu

PNG logo dosyasını bu klasöre yerleştirin:

**Dosya Adı:** `qkv_core_logo.png`

**Tam Yol:** `logo_designs/qkv_core_logo.png`

## Önerilen Boyutlar

Web UI için optimize edilmiş boyutlar:

- **Header Logo:** 300px genişlik (yükseklik otomatik)
- **Home Tab Logo:** 250px genişlik (yükseklik otomatik)
- **Favicon:** 32x32px veya 64x64px (opsiyonel)

## Logo Özellikleri

- **Format:** PNG (transparan arka plan)
- **Renkler:** RGB (Kırmızı-Q, Yeşil-K, Mavi-V)
- **Arka Plan:** Transparan
- **Kalite:** Yüksek çözünürlük (300 DPI önerilir)

## Kullanım

Logo dosyasını `logo_designs/qkv_core_logo.png` olarak kaydedin. Web UI otomatik olarak logoyu yükleyecektir.

Eğer logo dosyası bulunamazsa, web UI varsayılan metin başlığını kullanacaktır.
